Was ist ECMAScript?

Geschichte

  • 1995: JavaScript erscheint als LiveScript
  • 1997: ECMAScript Standard wird eingeführt
  • 1999: ES3 wird veröffentlicht und IE5 ist in Mode
  • 2000–2005: XMLHttpRequest, a.k.a. AJAX, gewinnt an Popularität in Anwendungen wie Outlook Web Access (2000) und Oddpost (2002), Gmail (2004) und Google Maps (2005)
  • 2009: ES5 wird veröffentlicht mit forEach, Object.keys, Object.create und standard JSON
  • 2015: ES6/ECMAScript2015 erscheint

Was ist ECMAScript?

Allgemeines

  • ECMA = European Computer Manufacturers Association
  • geschützte scripting-language Spezifikation
  • standardisiert von Ecma International
  • ECMA-262 und ISO/IEC 16262
  • wurde zur Standardisierung von JavaScript erstellt

    var amount = 0
